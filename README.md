# Notes app

## SPA application with React + Flux, NodeJS + Express and MongoDB.

### Installation

```sh
$ git clone https://github.com/AlexeyKagan/spa-notes-app.git
$ cd spa-notes-app
$ npm i
$ open mongodb on port 27017
$ npm run server
$ npm run webpack-devserver
$ type in browser: localhost:8090
```
